Welcome: EACL 2026 Personal Planning Workspace
Main goal: help participants explore papers, discover semantic clusters, build preference rankings, and turn those preferences into a practical conference plan.
Recommended workflow:
1) Start in Overview to understand papers in session context.
2) Use 2D Embeddings to discover clusters and similar papers.
3) Use Pairwise Ranking to learn your preferences quickly.
4) Use Oral Schedule to choose in-person room priorities.
5) Use Posters to get a ranked shortlist to visit.

Function Guide (all tabs and controls)
Top controls:
- Load CSV / Auto-load CSV: load conference program data.
- Reset ranking: clear pairwise preference history.
- Import/Export state: save/restore ranking progress.
- Export scored CSV: write rankings (`pref_mu`, `pref_rank`, uncertainty) back to CSV.

Overview tab:
- Navigate papers in Date → Session → Location hierarchy.
- Filter by search text, category, session, and location.
- Use this to inspect individual papers in their session context.

2D Embeddings tab:
- Visualize semantic clusters with PCA / t-SNE / UMAP.
- Use search highlight/filter, oral-only shortcut, and sampling.
- Click papers to inspect nearest neighbors and related titles.

Pairwise Ranking tab:
- Compare paper pairs with Active / Random / Bubble / Resolve-ties modes.
- Use A/B, Strong A/B, Both, Neither, Skip, Undo actions.
- Build a personalized preference ranking from lightweight decisions.

Oral Schedule tab:
- Produces primary + backup room suggestions per time slot.
- Combines learned ranking with topic-match signals.
- Focuses on in-person oral-session decision support.

Posters tab:
- Groups poster papers by date/time blocks.
- Sorts each block by your current rank to prioritize visits.
